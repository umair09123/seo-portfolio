Case Study

Technical SEO — Publisher Site Crawl

ClientPublisher Site (Canada)
IndustryMedia / Publishing
Duration2 Months
ServiceTechnical SEO

🔴 Problem

  • 40,000+ low-value URLs wasting crawl budget
  • Only 35% of pages indexed by Google
  • LCP (load time) 4.2 seconds — poor Core Web Vitals
  • Duplicate content across pagination and filters

🟡 Process

  • Log-file analysis to see Googlebot crawl patterns
  • No-indexed pagination and filter URLs
  • Optimized robots.txt to block low-value pages
  • Improved Core Web Vitals (image optimization, caching)
  • Submitted new sitemap to Google Search Console

🟢 Outcome

+71%Pages indexed
-2.1sLCP improvement
+35%Organic traffic

Indexation jumped from 35% to 70%. Load time reduced from 4.2s to 2.1s. Traffic improved significantly.

📘 Learning

  • Crawl budget is valuable — don't waste it
  • Log-file analysis reveals hidden crawl issues
  • Core Web Vitals directly impact rankings
  • Robots.txt and no-index are powerful tools
